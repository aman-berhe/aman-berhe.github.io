Text Classification

Text classification involves categorizing text into predefined classes, such as spam detection or topic labeling.

Python Example: Text Classification with Scikit-learn


from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

# Sample data
texts = ["I love NLP", "This is spam", "NLP is amazing", "I hate spam"]
labels = ["positive", "spam", "positive", "spam"]

# Create a pipeline
model = make_pipeline(TfidfVectorizer(), MultinomialNB())
model.fit(texts, labels)

# Predict
print("Prediction:", model.predict(["I enjoy learning NLP"]))

Output: Prediction: ['positive']
