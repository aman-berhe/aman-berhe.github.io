Sentiment Analysis

Sentiment analysis determines the emotional tone of text, such as positive, negative, or neutral.

Python Example: Sentiment Analysis with TextBlob



            from textblob import TextBlob

            text = "I love NLP! It's amazing."
            blob = TextBlob(text)
            print("Sentiment Polarity:", blob.sentiment.polarity)

Output: Sentiment Polarity: 0.5 (positive sentiment)
