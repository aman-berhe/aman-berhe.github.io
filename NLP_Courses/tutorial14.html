Text Summarization

Text summarization generates a concise summary of a longer text.

Python Example: Summarization with Hugging Face


from transformers import pipeline

summarizer = pipeline("summarization")
text = "Natural Language Processing is a fascinating field. It enables machines to understand and generate human language. NLP has applications in chatbots, sentiment analysis, and more."
summary = summarizer(text, max_length=30, min_length=10, do_sample=False)
print("Summary:", summary[0]['summary_text'])

Output: Summary: NLP is a fascinating field with applications in chatbots and sentiment analysis.
