Question Answering

Question answering systems retrieve answers to questions from a given context.

Python Example: QA with Hugging Face


            from transformers import pipeline

            qa_pipeline = pipeline("question-answering")
            context = "Natural Language Processing (NLP) is a field of AI focused on human language."
            question = "What is NLP?"
            result = qa_pipeline(question=question, context=context)
            print("Answer:", result['answer'])

Output: Answer: a field of AI focused on human language
