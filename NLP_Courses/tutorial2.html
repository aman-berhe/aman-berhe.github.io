Text Preprocessing

Text preprocessing is a crucial step in NLP that involves cleaning and preparing text data for analysis. Common steps include removing special characters, lowercasing, and handling whitespace.

Python Example: Cleaning Text


import re

text = "Hello! This is a sample text with numbers 123 and symbols #@$."
cleaned_text = re.sub(r'[^a-zA-Z\s]', '', text)  # Remove special characters and numbers
cleaned_text = cleaned_text.lower()  # Convert to lowercase
print("Cleaned Text:", cleaned_text)

Output: Cleaned Text: hello this is a sample text with numbers and symbols
